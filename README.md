# Servicio rest de Movimiento de cuentas BDP


Technologies Used

- Spring Boot 2.4.1
- Spring Data JPA
- Lombok
- MySQL

## How to Run this application

First generate a build:

```shell
$ ./gradlew clean build
```
or
```shell
$ gradle clean build
```

next, run with this command:

```shell
$ docker-compose build
```

```shell
$ docker-compose up
```

To stop the api

```shell
$ docker-compose down
```

if you want remove all images

```shell
$ docker-compose down -rmi all
```
